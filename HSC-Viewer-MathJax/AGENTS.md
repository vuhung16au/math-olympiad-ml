# HSC-Viewer-MathJax Agent Guide

## Stack

- Next.js App Router
- TypeScript
- bun
- MathJax
- make4ht
- Tailwind CSS
- Vercel

## Rules

1. Keep `.tex` files in sibling `HSC-xxx` folders unchanged.
2. Treat `.generated/` as derived output only.
3. Use `make` targets for standard workflows.
4. Keep docs short and practical.
5. Reuse the `HSC-Viewer` color palette and simple reader layout.
