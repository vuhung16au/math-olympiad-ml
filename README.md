# math-olympiad-ml

A collection of documents and findings on IMO-level mathematics and machine learning/AI topics (and some HSC Math Extension 2). This repository contains various mathematical problems, proofs, and computational solutions ranging from olympiad-level problems to advanced mathematical analysis.

## Sub-Projects

Below is a list of the main sub-projects in this repository. Each folder contains documents, code, or both, related to a specific mathematical or machine learning topic.

- **[DroneShow](./DroneShow/):** Computational and algorithmic explorations for drone light shows, including path planning and simulation.
- **[functional-equation-x2+x+1](./functional-equation-x2+x+1/):** LaTeX and code for functional equations involving $x^2 + x + 1$.
- **[HSC_sqrt_un](./HSC_sqrt_un/):** HSC Mathematics sequence proof project. Contains LaTeX documents and proofs related to sequence problems, including compiled PDF outputs.
- **[HSC-Collections](./HSC-Collections/):** A collection of HSC-level problems, solutions, and LaTeX documents.
- **[HSC-ComplexNumbers](./HSC-ComplexNumbers/):** Problems and solutions related to complex numbers at the HSC level.
- **[HSC-Induction](./HSC-Induction/):** Induction problems, proofs, and related resources.
- **[HSC-Inequalities](./HSC-Inequalities/):** Inequality problems, solutions, and LaTeX resources.
- **[HSC-Integrals](./HSC-Integrals/):** Integral calculus problems, solutions, and classification.
- **[HSC-Mechanics](./HSC-Mechanics/):** Mechanics problems, solutions, and supporting documents.
- **[HSC-Polynomials](./HSC-Polynomials/):** Polynomial problems and solutions.
- **[HSC-Proofs](./HSC-Proofs/):** General mathematical proofs and problem sets.
- **[HSC-Vectors](./HSC-Vectors/):** Vector algebra and geometry problems and solutions.
- **[Information-Geometry-for-GenerativeAI](./Information-Geometry-for-GenerativeAI/):** Notes and explorations on information geometry in generative AI.
- **[JacobianMathsToDeepLearning](./JacobianMathsToDeepLearning/):** Jacobian matrices and their applications in deep learning.
- **[Manifold](./Manifold/):** Manifold theory and its applications in mathematics and machine learning.
- **[Markov-Chain-Monte-Carlo](./Markov-Chain-Monte-Carlo/):** MCMC methods, notes, and code.
- **[MathsHeartShaped3D](./MathsHeartShaped3D/):** Mathematical modeling of heart-shaped 3D objects.
- **[MeanExpectationLimit](./MeanExpectationLimit/):** Problems and notes on means, expectations, and limits.
- **[NeuralNets-From-Math](./NeuralNets-From-Math/):** Mathematical foundations of neural networks.
- **[PappusTheorem](./PappusTheorem/):** Explorations and proofs related to Pappus's Theorem.
- **[quintic](./quintic/):** Quintic Equation Solver - A Python-based solver using SymPy to solve quintic equations (polynomial equations of degree 5). Demonstrates symbolic and numerical approaches to finding solutions for equations like $x^5 - 5x^3 + 5x - 4 = 0$.
- **[RationalEverywhereNowhere](./RationalEverywhereNowhere/):** Problems and notes on rational functions and their properties.
- **[sequence_a_n](./sequence_a_n/):** Problem T6-219 - A sequence analysis problem involving recursive sequences and limit inferior proofs. Contains problem statement (PDF), LaTeX source, and analysis notes.
- **[StrangeNestSquareRoots](./StrangeNestSquareRoots/):** Explorations of nested square root problems. See the [external repository](https://github.com/vuhung16au/MachineLearning-GenAI/tree/main/StrangeNestSquareRoots) for more details.
