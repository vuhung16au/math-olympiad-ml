# Problems for HSC Math Extension 2 Last Resorts

This folder contains the selected problems for Part 1 (detailed solutions) and Part 2 (less detailed solutions) of the Last Resorts project. Each problem is sourced from the samples and classified by difficulty.

- Easy: 1-3 reasoning steps
- Medium: 4+ reasoning steps, 2-3 concepts
- Hard: 8+ reasoning steps, 4+ concepts

See solutions in the /solutions/ folder.
