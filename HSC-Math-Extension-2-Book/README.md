This folder contains the combined HSC Mathematics Extension 2 book, which assembles the existing topic booklets into a single volume with part pages, a shared preface, and a table of contents.

## Structure

The publishable structure mirrors the clean subset of `HSC-LastResorts`:

- `HSC-Math-Extension-2-Book.tex` - main LaTeX source
- `fonts/` - shared font assets
- `problems/` - placeholder folder retained for structure parity
- `releases/` - compiled PDF output
- `solutions/` - placeholder folder retained for structure parity
- `styles/` - shared style files
