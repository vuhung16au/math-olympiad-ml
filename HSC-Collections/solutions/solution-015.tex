\subsection{Solution to Problem 15: Integer Equation with Large Exponents}

\begin{solution}
\textbf{Parity-only argument (no modular arithmetic).} Consider even and odd values of $n$ separately.

If $n$ is even, then $n+1$ is odd, so $(n+1)^{41}$ is odd; also $n^{40}$ is even, so $79n^{40}$ is even. Thus the left-hand side is odd minus even, which is \textbf{odd}.

If $n$ is odd, then $n+1$ is even, so $(n+1)^{41}$ is even; also $n^{40}$ is odd, so $79n^{40}$ is odd. Thus the left-hand side is even minus odd, which is again \textbf{odd}.

In every case the left-hand side is odd, while the right-hand side is $2$, which is even. An odd integer cannot equal an even integer, so there is no integer $n$ satisfying the equation.

\textit{Note.} Modular arithmetic is not in the syllabus, but it is a useful tool. For completeness, the same argument in mod $2$ is: in both cases above, $(n+1)^{41} - 79n^{40} \equiv 1 \pmod{2}$, whereas $2 \equiv 0 \pmod{2}$.
\end{solution}

\begin{takeaways}
\begin{itemize}
    \item Parity alone shows the left side is always odd while $2$ is even, so no solution exists
    \item Split into even $n$ and odd $n$; powers preserve parity in each case
    \item Modulo $2$ language is optional here but gives the same contradiction compactly
\end{itemize}
\end{takeaways}
