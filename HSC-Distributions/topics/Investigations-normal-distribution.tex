% Topic: Investigations Using the Normal Distribution

This section develops investigative skills with the normal distribution, including:

\begin{itemize}[leftmargin=*]
  \item estimating population parameters from sample data,
  \item testing whether a data set is approximately normally distributed (e.g.\ using a histogram or normal probability plot),
  \item interpreting results in context.
\end{itemize}

\textbf{Checking normality.} A data set may be approximately normal if:
\begin{itemize}[leftmargin=*]
  \item its histogram is roughly bell-shaped and symmetric,
  \item approximately $68\%$, $95\%$, and $99.7\%$ of data lie within 1, 2, and 3 standard deviations of the mean respectively.
\end{itemize}
