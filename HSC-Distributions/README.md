# HSC-Distributions

**HSC Mathematics Extension 2: Probability Distributions**

A booklet covering probability distributions for NSW HSC Mathematics Extension 2 students, including continuous distributions, the normal distribution, binomial distributions, and advanced topics.

## Topics Covered

### Continuous Probability Distributions
- Relative frequency
- Continuous distributions
- Mean and variance of a distribution
- The standard normal distribution
- General normal distributions
- Applications of the normal distribution
- Investigations using the normal distribution

### Binomial Distributions
- Binomial probability
- Binomial distributions
- Normal approximations to a binomial
- Sample proportions

### Advanced Topics (outside HSC syllabus)
- Monte Carlo simulations
- Hypergeometric distribution

## Building

Requires Docker with a TeX Live image.

```bash
make pdf      # Build the PDF
make open     # Open the PDF
make clean    # Remove build artefacts
make release  # Copy PDF to releases/
make final    # Clean, build, and release
```
