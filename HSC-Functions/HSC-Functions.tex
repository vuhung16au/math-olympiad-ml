\section{Introduction}
% ============================================================

\subsection{Project Overview}
This booklet is a growing collection of functions problems for NSW HSC Mathematics Extension~1 students, with a few short Extension~2 preview notes where they help build mathematical maturity. It covers the major themes of the syllabus: understanding what a function is, working with inverses, composing functions, and analysing their graphs. The problems are designed to build fluency with algebraic manipulation and geometric reasoning side by side.

\subsection{Target Audience}
This resource is written for students who want extra practice with functions in a clear, classroom-friendly format. Solutions are intentionally concise, but each one highlights the key idea---whether that is a domain restriction, a symmetry argument, or a composition rule. The main focus is Extension~1, while some comments point ahead to Extension~2 or university mathematics.

\subsection{How to Use This Booklet}
\begin{itemize}[leftmargin=*]
  \item Read the short summary of the core definitions and ideas first.
  \item Attempt each problem before reading the solution.
  \item Pay attention to domains and ranges: they are often where mistakes happen.
  \item Look for symmetry: the graph of an inverse is the reflection of the original in the line $y=x$.
  \item Use the takeaways to build pattern recognition for future HSC questions.
\end{itemize}

\subsection{Extension 1 Knowledge Needed}
Most problems in this booklet are accessible with standard HSC Mathematics Extension~1 knowledge. The key background ideas are:

\begin{itemize}[leftmargin=*]
  \item \textbf{Function notation:} writing $f(x)$, understanding that $f$ is the rule and $f(x)$ is the value, and reading expressions such as $f(a+h)$ or $f(f(x))$.
  \item \textbf{Domain and range:} identifying the largest possible domain (the natural domain) and the corresponding range, and applying domain restrictions when stated.
  \item \textbf{One-to-one functions:} recognising that a function has an inverse if and only if it passes the horizontal line test.
  \item \textbf{Inverse functions:} swapping $x$ and $y$ to find $f^{-1}$, and understanding that $f(f^{-1}(x))=x$ and $f^{-1}(f(x))=x$.
  \item \textbf{Graphs of inverse functions:} reflecting the graph of $f$ in the line $y=x$ to obtain $f^{-1}$.
  \item \textbf{Composite functions:} forming $f(g(x))$ by substituting one function into another, and identifying the correct domain of the composition.
  \item \textbf{Even and odd functions:} $f$ is even if $f(-x)=f(x)$ for all $x$ (the graph has $y$-axis symmetry); $f$ is odd if $f(-x)=-f(x)$ for all $x$ (the graph has rotational symmetry about the origin).
  \item \textbf{Transformations:} translations, reflections, dilations, and their effect on the equation and graph of a function.
\end{itemize}

\subsection{Core Language of Functions}

\textbf{Function.} A function $f: A \to B$ assigns to every element $x \in A$ exactly one element $f(x) \in B$. The set $A$ is the \textbf{domain}, the set $B$ is the \textbf{codomain}, and the \textbf{range} is the set of outputs that actually occur:
\[
\operatorname{range}(f) = \{f(x) : x \in A\} \subseteq B.
\]

\textbf{The empty function.} If the domain is the empty set $\varnothing$, then there is exactly one function
\[
f:\varnothing \to B,
\]
called the \textbf{empty function}. It has no inputs and therefore no outputs. Its range is also empty. This is mainly a set-theoretic idea, but it is good to know that it is still a perfectly valid function.

\textbf{Variable.} A variable is a symbol, such as $x$ or $t$, that stands for a value that may change. In $y=f(x)$, the variable $x$ is the input and the variable $y$ is the output.

\textbf{Map or mapping.} A map is another name for a function or relation that pairs elements of one set with elements of another set.

\subsection{Relations and Mappings}

A \textbf{relation} between sets $A$ and $B$ is any collection of ordered pairs $(a,b)$ with $a \in A$ and $b \in B$. A function is a special kind of relation in which each input has exactly one output.

The most common mapping types are shown below.

One-to-one and many-to-one relations can be functions. One-to-many and many-to-many relations are not functions because at least one input has more than one output.

\subsection{Domain, Range, and Basic Tests}

\textbf{Domain.} The domain is the set of allowed input values.

\textbf{Range.} The range is the set of actual output values.

\textbf{Vertical line test.} A curve in the plane is the graph of a function if and only if every vertical line meets the curve at most once.

\textbf{Horizontal line test.} A function is one-to-one (injective) if and only if every horizontal line meets its graph at most once. Only one-to-one functions have an inverse that is also a function.

\subsection{Important Number Sets}

We often write:
\[
\mathbb{N} = \{1,2,3,\dots\}, \qquad
\mathbb{Z} = \{\dots,-2,-1,0,1,2,\dots\},
\]
\[
\mathbb{Q} = \left\{\frac{a}{b} : a,b \in \mathbb{Z},\, b \ne 0\right\}, \qquad
\mathbb{R} = \text{the set of real numbers}.
\]

Good to know for later: the set of complex numbers is written $\mathbb{C}$, not $\mathbb{Z}$. Complex numbers are studied in Extension~2, so in this booklet they appear only as short preview notes.

\subsection{Important Families of Functions}

\textbf{Linear function.} A linear function has the form
\[
f(x)=mx+c,
\]
where $m$ is the gradient and $c$ is the $y$-intercept. Its graph is a straight line.

\textbf{Quadratic function.} A quadratic function has the form
\[
f(x)=ax^2+bx+c, \qquad a \ne 0.
\]
Its graph is a parabola.

\textbf{Polynomial.} A polynomial over $\mathbb{R}$ is an expression of the form
\[
P(x)=a_nx^n+a_{n-1}x^{n-1}+\dots+a_1x+a_0,
\]
where the coefficients $a_0,a_1,\dots,a_n$ are real numbers. If the coefficients are allowed to be complex numbers, then $P$ is a polynomial over $\mathbb{C}$. This is a good-to-know Extension~2 idea.

\textbf{Cubic function.} A cubic function has the form
\[
f(x)=ax^3+bx^2+cx+d, \qquad a \ne 0.
\]

\textbf{Reciprocal function.} A basic reciprocal function is
\[
f(x)=\frac{1}{x},
\]
which is undefined at $x=0$.

\subsection{Continuity and Differentiation Preview}

\textbf{Continuity.} Informally, a function is continuous if its graph can be drawn without lifting your pen. More precisely, $f$ is continuous at $x=a$ if small changes in $x$ produce small changes in $f(x)$.

\textbf{Continuous function.} A function is continuous on an interval if it is continuous at every point of that interval.

\textbf{Extension~2 preview.} In more formal Extension~2 language, $f$ is continuous at $x=a$ if
\[
\lim_{x \to a} f(x) = f(a).
\]

\textbf{Derivative.} The derivative $f'(x)$ measures the instantaneous rate of change of $f$ and the gradient of the tangent to the graph.

\textbf{Differentiation.} Differentiation is the process of finding the derivative. This is Extension~2 material, but it is good to know because many important functions are later studied through their derivatives.

\textbf{Sequence of numbers.} A sequence is an ordered list of numbers, usually written
\[
a_1,a_2,a_3,\dots
\]
or as $(a_n)$, where $n$ is a positive integer.

\textbf{Limit of a sequence.} We say $(a_n)$ has limit $L$ if the terms get closer and closer to $L$ as $n$ becomes very large. We write
\[
\lim_{n\to\infty} a_n = L.
\]

\textbf{Why this matters for functions.} Sequences help us understand what a function does when $x$ approaches a value or becomes very large. They are one of the basic ideas behind limits, continuity, and later calculus. This is mainly Extension~2 and university material, but it is very useful to know early.

\subsection{Symmetry, Inverses, and Composition}

\textbf{Even and odd functions.}
\[
f \text{ is even} \iff f(-x) = f(x) \text{ for all } x.
\qquad
f \text{ is odd} \iff f(-x) = -f(x) \text{ for all } x.
\]
An even function has symmetry about the $y$-axis. An odd function has rotational symmetry about the origin.

\textbf{Inverse function.} If $f$ is one-to-one with domain $A$ and range $B$, then its inverse $f^{-1}: B \to A$ satisfies
\[
f^{-1}(f(x)) = x \quad \text{for all } x \in A,
\qquad
f(f^{-1}(y)) = y \quad \text{for all } y \in B.
\]
To find $f^{-1}$: write $y = f(x)$, swap $x$ and $y$, then solve for $y$.

\textbf{Reflection principle.} The graph of $f^{-1}$ is the reflection of the graph of $f$ in the line $y = x$.

\textbf{Composite function.} Given $f: B \to C$ and $g: A \to B$, the composite $f \circ g: A \to C$ is defined by
\[
(f \circ g)(x) = f(g(x)).
\]
Note that in general $f \circ g \neq g \circ f$.

\textbf{Domain of a composite.} For $f(g(x))$ to be defined, $x$ must be in the domain of $g$ \emph{and} $g(x)$ must be in the domain of $f$:
\[
\operatorname{dom}(f \circ g) = \{x \in \operatorname{dom}(g) : g(x) \in \operatorname{dom}(f)\}.
\]

\textbf{Fixed point.} A fixed point of a function $f$ is a value $a$ such that
\[
f(a)=a.
\]
Graphically, fixed points are the points where the graph of $y=f(x)$ meets the line $y=x$.

\subsection{Key Transformations}

Given a base function $y = f(x)$, the standard transformations are:

\begin{itemize}[leftmargin=*]
  \item $y = f(x) + c$: shift up by $c$ (down if $c < 0$).
  \item $y = f(x - h)$: shift right by $h$ (left if $h < 0$).
  \item $y = af(x)$: vertical dilation by factor $|a|$; reflection in $x$-axis if $a < 0$.
  \item $y = f(bx)$: horizontal dilation by factor $\tfrac{1}{|b|}$; reflection in $y$-axis if $b < 0$.
  \item Replacing $y$ by $-f(x)$ reflects the graph in the $x$-axis.
  \item Replacing $x$ by $-x$ gives $y=f(-x)$, which reflects the graph in the $y$-axis.
  \item Rotations can also be studied, but they are less central in HSC Extension~1 function questions because a rotated graph need not remain a function.
\end{itemize}

These transformations combine: $y = af(bx - h) + c$ is the general form.

\subsection{Piecewise-Defined Functions}
A piecewise function assigns a different rule to different parts of the domain. For example:
\[
f(x) = \begin{cases} x^2 & x < 0, \\ 2x+1 & x \ge 0. \end{cases}
\]
Key skills are: evaluating at a given $x$, finding the range piece by piece, and drawing the graph with correct open/closed endpoints.

\subsection{Absolute Value as a Piecewise Function}
\[
|x| = \begin{cases} x & x \ge 0, \\ -x & x < 0. \end{cases}
\]
The graph of $y = |f(x)|$ reflects any portion of $y = f(x)$ that lies \emph{below} the $x$-axis up above it. The graph of $y = f(|x|)$ keeps the right half of $y = f(x)$ and reflects it to create an even function.

\textbf{Absolute value of a real number.} For a real number $x$, the absolute value $|x|$ is its distance from $0$ on the number line.

\textbf{Absolute value of a complex number.} Good to know for Extension~2 and beyond: if $z=a+bi$, then
\[
|z|=\sqrt{a^2+b^2}.
\]
This is the distance of $z$ from the origin in the complex plane. Complex numbers belong to Extension~2, not Extension~1, so this booklet only mentions them briefly.

\subsection{Reciprocals, Asymptotes, and Sign Tables}

\textbf{Reciprocal of a function.} The reciprocal of $f(x)$ is
\[
\frac{1}{f(x)},
\]
provided that $f(x) \ne 0$.

\textbf{Asymptotes.} An asymptote is a line that the graph approaches. For example, $y=\frac{1}{x}$ has vertical asymptote $x=0$ and horizontal asymptote $y=0$.

\textbf{Table of signs.} A table of signs records where an expression is positive, zero, or negative. It is especially useful for rational functions, inequalities, and sketching graphs.

\subsection{Parametric and Multivariable Functions}

\textbf{Parametric definition.} Sometimes a relation is defined by a third variable, called a parameter. For example,
\[
x=t, \qquad y=\frac{1}{t}, \qquad t \ne 0.
\]
This defines the curve $y=\frac{1}{x}$ in parametric form.

\textbf{Multiple-variable functions.} A function can also take more than one input, such as
\[
f(x,y)=x^2+y^2.
\]
These are important in higher mathematics but are outside the main Extension~1 and Extension~2 syllabuses.

\subsection{Important HSC Functions}

The most important functions to recognise quickly in HSC mathematics are:

\begin{itemize}[leftmargin=*]
  \item linear functions $mx+c$,
  \item quadratic functions $ax^2+bx+c$,
  \item cubic functions,
  \item reciprocal functions such as $\frac{1}{x}$ and more general rational functions,
  \item absolute value functions,
  \item exponential functions such as $a^x$ with $a>0$ and $a \ne 1$,
  \item logarithmic functions such as $\log_a x$,
  \item trigonometric functions such as $\sin x$, $\cos x$, and $\tan x$.
\end{itemize}

In Extension~1, the main emphasis is on algebraic, exponential, logarithmic, and trigonometric functions. In Extension~2, ideas such as continuity, differentiation, and later complex-number-based viewpoints become more important.

\subsection{A Brief Note on Big $O$ Notation}
Big $O$ notation is a way of comparing how fast functions grow. It is widely used in university mathematics and computer science.

If we write
\[
f(x)=O(g(x)) \quad \text{as } x\to\infty,
\]
we mean that for sufficiently large $x$, the function $f(x)$ grows no faster than a constant multiple of $g(x)$.

This topic is \textbf{outside the HSC Mathematics Extension~1 and Extension~2 syllabuses}, but it is very good to know for later university studies.

\textbf{Examples.}
\begin{itemize}[leftmargin=*]
  \item If $f(x)=3x^2+5x+7$, then $f(x)=O(x^2)$ because the $x^2$ term dominates for large $x$.
  \item If $f(x)=x^5-2x^3+1$, then $f(x)=O(x^5)$.
  \item If $f(x)=\log x$, then $f(x)=O(\log x)$.
  \item If $f(x)=2^x+100x^4$, then $f(x)=O(2^x)$ because the exponential term eventually dominates the polynomial term.
\end{itemize}

\textbf{Growth comparison to remember.} As $x\to\infty$,
\[
\log x \text{ grows more slowly than any positive power of } x,
\]
\[
x^n \text{ grows more slowly than } a^x \quad \text{for any fixed } n\in\mathbb{N} \text{ and } a>1.
\]
So in rough order of growth:
\[
\log x \quad < \quad x,\ x^2,\ x^3,\dots \quad < \quad 2^x,\ 3^x,\dots
\]

\textbf{Polynomial example.} If
\[
P(x)=a_nx^n+a_{n-1}x^{n-1}+\dots+a_0
\]
with $a_n\ne 0$, then
\[
P(x)=O(x^n).
\]
So the degree of the polynomial tells you its dominant growth rate.

\subsection{Problem-Solving Habits for This Booklet}
\begin{itemize}[leftmargin=*]
  \item Always state the domain and range of your answer, not just the rule.
  \item When finding an inverse, restrict the domain of $f$ first if the function is not one-to-one.
  \item When composing, work from the inside out: compute $g(x)$ first, then apply $f$.
  \item Sketch the graph whenever the question involves domains, ranges, inverses, or asymptotes.
  \item In even/odd questions, substitute $-x$ and simplify fully before drawing a conclusion.
  \item For rational expressions, locate any zeroes and undefined points before making a sign table.
\end{itemize}

\subsection{A Small Extension~2 Preview}
This booklet is mainly about HSC Mathematics Extension~1. When complex numbers, formal limits, or derivatives are mentioned, they should be read as \textbf{good to know} ideas that will appear properly in Extension~2 or in later university mathematics.

\textbf{Continuity in formal language.} For Extension~2 students, a function is \textbf{continuous} at $x = a$ if
\[
\lim_{x \to a} f(x) = f(a).
\]
Piecewise functions need this checked at the break-points. Many Extension~2 questions later ask you to verify continuity or use derivatives to study turning points and behaviour.

\begin{remark}
The most common functions mistake is forgetting to adjust the domain when restricting a function to make it one-to-one before finding its inverse. Always check that your inverse has the correct domain and range.
\end{remark}

\begin{remark}[Notation warning]
$f^{-1}(x)$ means the inverse \emph{function}, not $\dfrac{1}{f(x)}$. If you mean the reciprocal, write $[f(x)]^{-1}$ or $\dfrac{1}{f(x)}$ explicitly.
\end{remark}

\subsection{Strict Definitions of Continuity}

These strict definitions are very helpful for higher studies and for understanding mathematics, functions, and continuity in a rigorous way.

\textbf{1. The limit definition (calculus).} A function $f(x)$ is continuous at a point $x=c$ if and only if all three conditions hold:
\begin{itemize}[leftmargin=*]
  \item $f(c)$ exists.
  \item $\lim_{x\to c}f(x)$ exists (that is, the left-hand and right-hand limits both exist and are equal).
  \item $\lim_{x\to c}f(x)=f(c)$.
\end{itemize}

If even one condition fails at $x=c$ (for example, a hole, a jump, or a vertical asymptote), then $f$ is discontinuous at $c$.

\textbf{2. The $\epsilon$-$\delta$ definition (real analysis).} Let $f$ be defined on a domain $D$, and let $c\in D$. We say $f$ is continuous at $c$ if:
\begin{quote}
For every real number $\epsilon>0$, there exists a real number $\delta>0$ such that for all $x\in D$,
\[
|x-c|<\delta \implies |f(x)-f(c)|<\epsilon.
\]
\end{quote}

This is the most rigorous standard definition of continuity. It replaces informal language like ``$x$ approaches $c$'' with precise inequalities.

\subsection{Open and Closed Intervals}

An interval is \emph{closed} if it contains all of its endpoints, and \emph{open} if it does not contain any of its endpoints.

\textbf{Conceptual Examples:}
\begin{enumerate}[label=\textbf{\alph*.}]
  \item \textbf{Explain why the degenerate interval $[5, 5]$ is closed.} \\
  The interval $[5, 5]$ contains exactly one real number: $5$. Its endpoints are $5$ and $5$. Since the interval contains $5$, it contains all of its endpoints, meeting the exact definition of a closed interval.
  
  \item \textbf{Explain why the interval $(-\infty, \infty)$ is open.} \\
  An interval is open if it does not contain any endpoints. The interval $(-\infty, \infty)$, representing the entire real line $\mathbb{R}$, has no boundaries or endpoints in the real numbers. Because there are no endpoints to contain, it vacuously satisfies the condition of not containing any of its endpoints.
  
  \item \textbf{Explain why the interval $(-\infty, \infty)$ is closed.} \\
  An interval is closed if it contains \emph{all} of its endpoints. Since it has zero endpoints, there are no endpoints missing from the set. Therefore, it vacuously contains all of its endpoints. In topology, spaces with this property are considered both open and closed (clopen).
\end{enumerate}
