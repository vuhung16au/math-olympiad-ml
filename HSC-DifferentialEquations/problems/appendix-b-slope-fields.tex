When reading a slope field, focus on the pattern before trying to draw a full solution curve.

\begin{enumerate}[leftmargin=*]
  \item \textbf{Look for flat segments.} Wherever the slope is zero, solution curves have horizontal tangents. A whole horizontal band of zero slopes often signals an equilibrium solution.
  \item \textbf{Check sign regions.} If most segments in a region tilt upward, then solutions increase there. If they tilt downward, solutions decrease there.
  \item \textbf{Track steepness.} Steeper segments indicate larger magnitude of derivative, so the solution changes more rapidly.
  \item \textbf{Use the initial point carefully.} A particular solution must pass through the given starting point and then follow the local direction pattern.
  \item \textbf{Think about end behavior.} If nearby segments steer a curve toward a horizontal line, the solution may approach an equilibrium as $x \to \infty$ or $x \to -\infty$.
\end{enumerate}

\subsubsection*{Typical observations}
\begin{itemize}[leftmargin=*]
  \item If slopes depend only on $y$, then rows often repeat horizontally.
  \item If slopes depend only on $x$, then columns often repeat vertically.
  \item If the field changes sign above and below an equilibrium, that equilibrium may behave like an attracting or repelling level.
\end{itemize}

\subsubsection*{Examples worth recognizing}
\begin{itemize}[leftmargin=*]
  \item For $y'=-x-y$, the line $y=-x$ is a zero-slope isocline.
  \item For $y'=xy$, the coordinate axes form zero-slope guide lines because the derivative vanishes whenever $x=0$ or $y=0$.
  \item For $yy'+x=0$, rewriting as $y'=-\dfrac{x}{y}$ shows immediately that the field is undefined on $y=0$.
\end{itemize}

\subsubsection*{Isoclines}
An \textbf{isocline} is a curve along which the slope field has constant slope. If
\[
\frac{dy}{dx}=f(x,y),
\]
then setting $f(x,y)=m$ gives all points where the tiny direction segments have slope $m$.

For example, if
\[
y'=-x-y,
\]
then the zero-slope isocline is
\[
-x-y=0
\qquad \Longrightarrow \qquad
y=-x.
\]
Isoclines are a fast way to organize a slope field before sketching particular solution curves.

\subsubsection*{Common mistakes}
\begin{itemize}[leftmargin=*]
  \item Drawing a curve that cuts across the local segment directions.
  \item Forgetting that a slope field shows tangents, not disconnected graph pieces.
  \item Giving only a sketch when the question also asks for a verbal description of the long-run behavior.
\end{itemize}
