\section{Introduction}

\subsection{Project Overview}
This booklet is a growing collection of probability distribution problems for NSW HSC Mathematics Extension~2 students. It covers continuous probability distributions (including the normal distribution), binomial distributions, and selected advanced topics such as Monte Carlo simulations and the hypergeometric distribution.

\subsection{Target Audience}
This resource is written for students who want extra practice with probability distributions in a clear, classroom-friendly format. Full worked solutions are provided for Part~1 problems, with hints for Part~2 problems. Each problem contains a takeaways box summarising the key idea.

\subsection{How to Use This Booklet}
\begin{itemize}[leftmargin=*]
  \item Read the introductory theory for each topic before attempting problems.
  \item Attempt each problem before reading the solution.
  \item Pay attention to whether a question involves a discrete or continuous distribution.
  \item Use the takeaways to build pattern recognition for future HSC questions.
\end{itemize}

\subsection{Extension 1 Knowledge Needed}
To work through this booklet, students should be comfortable with:
\begin{itemize}[leftmargin=*]
  \item \textbf{Basic probability:} sample spaces, complementary events, conditional probability.
  \item \textbf{Algebra:} solving equations, working with exponentials and logarithms.
  \item \textbf{Calculus:} integration, including integration by substitution.
  \item \textbf{Binomial theorem:} expansion of $(1+x)^n$ and binomial coefficients $\binom{n}{r}$.
  \item \textbf{Statistics basics:} mean, variance, standard deviation.
\end{itemize}

\subsection{Core Formulas and Ideas}

\textbf{Probability density function (pdf).} A function $f(x)$ is a probability density function if
\[
f(x) \geq 0 \quad \text{for all } x, \qquad \text{and} \qquad \int_{-\infty}^{\infty} f(x)\,dx = 1.
\]
The probability that $X$ lies in the interval $[a,b]$ is
\[
P(a \leq X \leq b) = \int_a^b f(x)\,dx.
\]

\textbf{Mean (expected value).} For a continuous random variable with pdf $f(x)$,
\[
\mu = E(X) = \int_{-\infty}^{\infty} x\,f(x)\,dx.
\]

\textbf{Variance.}
\[
\sigma^2 = \mathrm{Var}(X) = E(X^2) - [E(X)]^2 = \int_{-\infty}^{\infty} x^2 f(x)\,dx - \mu^2.
\]

\textbf{Standard deviation.} $\sigma = \sqrt{\mathrm{Var}(X)}$.

\textbf{Standard normal distribution.} The standard normal distribution $Z \sim N(0,1)$ has pdf
\[
\phi(z) = \frac{1}{\sqrt{2\pi}}\,e^{-z^2/2}.
\]

\textbf{General normal distribution.} If $X \sim N(\mu, \sigma^2)$, then
\[
Z = \frac{X - \mu}{\sigma} \sim N(0,1).
\]

\textbf{Binomial distribution.} If $X \sim B(n, p)$, then
\[
P(X = r) = \binom{n}{r} p^r (1-p)^{n-r}, \quad r = 0, 1, \ldots, n.
\]
The mean is $\mu = np$ and the variance is $\sigma^2 = np(1-p)$.

\textbf{Normal approximation to binomial.} For large $n$,
\[
X \sim B(n,p) \approx N\!\left(np,\, np(1-p)\right).
\]

\textbf{Sample proportion.} For a sample of size $n$ from a population with proportion $p$,
\[
\hat{p} = \frac{X}{n}, \qquad E(\hat{p}) = p, \qquad \mathrm{Var}(\hat{p}) = \frac{p(1-p)}{n}.
\]

\section{Conclusion}

Probability distributions are at the heart of statistics and data science. The techniques covered in this booklet --- from reading a normal distribution table to applying the binomial model --- appear regularly in HSC examinations and in real-world applications such as quality control, risk analysis, and machine learning.

Keep practising with a variety of problems, and remember: understanding the shape and parameters of a distribution is often the key to unlocking the solution.
